Notes for PowerCycle


--------------------

All OSes


BIOS set to reboot after power loss
Disable any firewall
Setup a static IP address (example: 172.31.107.200)
The connection and IP must be available after boot completes and before login
--------------------

Windows OSes


Disable Windows Startup Repair
bcdedit /set {default} recoveryenabled No
bcdedit /set {current} bootstatuspolicy ignoreallfailures
Change Windows Startup Settings (F8?)
Change the default OS and the OS choice delay timer
